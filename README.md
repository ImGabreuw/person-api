## 💻 Prerequisites

* JDK 11
* Intellij IDEA

* Java 11
* Spring
  * JPA
  * Validation
  * Web
  * DevTools
  * pActuator
* MapStruct
* H2
* Lombok


## 🚀 Installation

```bash
# Repository clone
$ git clone https://github.com/ImGabreuw/person-api

# Go to the application directory
$ cd person-api/

# Start the application
$ mvn spring-boot:run
```

Go to http://localhost:8080/api/v1/person
